# 🍕 Pizza & Pasta Cafe Web App

Веб-приложение для кафе, позволяющее пользователям просматривать меню, оформлять заказы с доставкой и бронировать столики онлайн.

---

## 🚀 Возможности проекта

- 📖 Просмотр меню (пицца, паста, напитки и т.д.)
- 🛒 Оформление заказа с доставкой
- 🪑 Бронирование столиков онлайн

---

## 🧰 Используемые технологии

### Frontend:
- React
- TypeScript
- JavaScript
- HTML / CSS
- Node.js

### Backend:
- Python (FastAPI)

### База данных:
- SQLite3

---

## ⚙️ Инструкция по запуску

### 📌 Требования
Убедитесь, что установлен **Node.js**:
node -v

Скачать: https://nodejs.org

---

__Перейдите в корневую директорию проекта и выполните команды в терминале:__
Установить зависимости (один раз):
cd /путь/к/проекту

npm install

__Запустить фронтенд:__
npm run dev

После этого в терминале появится адрес вида http://localhost:5173 — откройте его в браузере.

---

__Если вы хотите запустить бэкенд (FastAPI):__
1) cd backend
2) python -m venv venv (тут могут быть проблемы, пробуйте разные варианты вместо python если не работает: py, python3)
3) ./venv/Scripts/activate
4) python -m pip install -r requirements.txt
5) uvicorn main:app --reload

Бэкенд запустится на http://localhost:8000
