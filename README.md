# 🍕 Pizza & Pasta Café Web App

Веб-приложение для кафе "Pizza & Pasta", позволяющее пользователям просматривать меню, оформлять заказы с доставкой и бронировать столики онлайн.

---

## 🚀 Возможности проекта

- 📖 Просмотр меню (пицца, паста, напитки и т.д.)
- 🛒 Оформление заказа с доставкой
- 🪑 Бронирование столиков онлайн

---

## 🧰 Используемые технологии

### Frontend:
- React
- TypeScript
- JavaScript
- HTML / CSS

### Backend:
- Node.js
- Python

### База данных:
- SQLite3

## __Инструкция по запуску__
Убедитесь, что у вас установлен Node.js (nodejs.org). Для проверки используйте:
node -v

---

__Перейдите в корневую директорию проекта и выполните команды в терминале:__
Установить зависимости (один раз):
cd /путь/к/проекту
npm install

__Запустить фронтенд:__
npm run dev

После этого в терминале появится адрес вида http://localhost:5173 — откройте его в браузере.

---

__Если вы хотите запустить бэкенд (FastAPI):__
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Если выдаёт ошибку на "uvicorn main:app --reload", то попробуйте:
python -m pip install -r requirements.txt

Бэкенд запустится на http://localhost:8000
