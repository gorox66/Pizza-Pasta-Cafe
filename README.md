🍕 Кафе доставки "Паста и Пицца"
Веб-приложение для заказа итальянских блюд с доставкой и бронированием столов в ресторане. Проект включает интерактивное меню, корзину, оформление заказа, личный кабинет с регистрацией и авторизацией, а также систему онлайн-бронирования столиков.

---

Инструкция по запуску
Убедитесь, что у вас установлен Node.js (nodejs.org). Для проверки используйте:

node -v

---

Перейдите в корневую директорию проекта и выполните команды в терминале:
Установить зависимости (один раз):
cd /путь/к/проекту
npm install

Запустить фронтенд:
npm run dev

После этого в терминале появится адрес вида http://localhost:5173 — откройте его в браузере.

---

Если вы хотите запустить бэкенд (FastAPI):
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Если выдаёт ошибку на "uvicorn main:app --reload", то попробуйте:
python -m pip install -r requirements.txt

Бэкенд запустится на http://localhost:8000
